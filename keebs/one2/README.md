# Ducky One 2

## Keybinds

| Keystroke  | Function          |
| ---------- | ----------------- |
| Fn+1-6     | Profiles          |
| Fn+F10     | LED Mode          |
| Fn+NumLock | Toggle mouse mode |

## Record Macro

**This doesn't work on Profile 1**

- Fn+Ctrl for 3 seconds
- Press bound key
- Press target key(s)
- Fn+Ctrl to finish
